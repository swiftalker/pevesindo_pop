# **3. Role Operasional Harian - Tugas & Kemampuan**

Bagian ini mendefinisikan siapa yang bertugas, apa tanggung jawabnya, dan apa yang dapat dilakukan di Center-App.

## **3.1 CSSR - Customer Sales Service Representative**

| **Aspek**       | **Detail**                                                                                          |
| --------------- | --------------------------------------------------------------------------------------------------- |
| **Tugas**       | Melayani customer (walk-in & online), input penjualan                                               |
| **KPI**         | Target bulanan                                                                                      |
| **Aksi di App** | Input penjualan umum & proyek, input CRM, assign survey ke Surveyor, notifikasi ke petugas lapangan |

## **3.2 Surveyor**

| **Aspek**       | **Detail**                                                                                                    |
| --------------- | ------------------------------------------------------------------------------------------------------------- |
| **Tugas**       | Survey lokasi, input laporan survey, pembuatan RAB, pengawasan proyek                                         |
| **KPI**         | Target bulanan, pengawasan proyek                                                                             |
| **Aksi di App** | Fulfil survey task, input laporan survey, input kebutuhan proyek, pembuatan RAB, delegasi ke Teknisi Lapangan |

## **3.3 Teknisi Lapangan**

| **Aspek**       | **Detail**                                                                                   |
| --------------- | -------------------------------------------------------------------------------------------- |
| **Tugas**       | Mengerjakan proyek sesuai blueprint, mematuhi protokol lapangan, input laporan proyek        |
| **KPI**         | Penyelesaian proyek                                                                          |
| **Aksi di App** | Input laporan proyek (milestone & progress), input kebutuhan & pengeluaran proyek ke Finance |

## **3.4 Warehouse Operator**

| **Aspek**       | **Detail**                                                                                                                               |
| --------------- | ---------------------------------------------------------------------------------------------------------------------------------------- |
| **Tugas**       | Persiapan pengiriman & penerimaan barang, pengawasan gudang, procurement                                                                 |
| **KPI**         | Kerapian administrasi gudang, pengiriman & penerimaan barang                                                                             |
| **Aksi di App** | Persiapan pengiriman & penerimaan, procurement, intervensi pergerakan barang, notifikasi ke Surveyor & CSSR, delegasi ke Helper & Driver |

## **3.5 Driver & Helper**

| **Role**   | **KPI**            | **Aksi di Center-App**                                                      |
| ---------- | ------------------ | --------------------------------------------------------------------------- |
| **Driver** | Kendaraan & barang | Assign pengiriman, laporan barang sampai, berita acara & odometer kendaraan |
| **Helper** | Kerapian gudang    | Terima & jalankan semua tugas yang di-assign, laporan tugas dan kegiatan    |
